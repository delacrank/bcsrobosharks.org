Directives and Expressions - Angular Js Development

Directives and Expressions

Directives allow us to extend HTML element attributes, expressions allow us to combine our directives in order to solve equations.

Angular Directives

Declaring variables in angular js is a little different then using just plain javascript. We will using 'ng-init' to intialize a html attribute to a value, let's look at an example below.

Code: 
<div ng-app="" ng-init="firstName='John'">
    <p>The name is <span ng-bind="firstName"></span></p>
</div>

Angular Js Computing Numbers

Remeber when we used our element attribute to define a variable. Let's provide values to those variables and then compute them in either an expression or by using data bind.

Code: 
<div ng-app="" ng-init="quantity=1;cost=5"> 
    // Expression    
    <p>Total in dollar: {{ quantity * cost }}</p>
    
    // Data Bind 
    <p>Total in dollar: <span ng-bind="quantity * cost"> </p>    
</div>

Angular Objects

Just like numbers we can also declare strings or objects. And then use them with our data bind or expression.

Code: 
<div ng-app="" ng-init="person={firstName:'Juan', lastName:'Carlos'}">
    <p>The name is {{ person.lastName }}</p>
    <p>The name is <span ng-bind="person.lastName"> </span> </p>
</div>

Arrays

Let's see how we can use the same methodology to add arrays.

Code: 
<div ng-app="" ng-init="points=[1,15,19,2,40]">
    <p>The third result is {{ points[2] }}</p>
</div>

Using Repeat

Using the angular repeat function is a way to call many elements from a collection similar to an array or an object.

Code: 
<div ng-app="" ng-init="names=['Jani','Hege','Kai']">
  <ul>
    <li ng-repeat="x in names">
            {{ x }}
        </li>
    </ul>
</div>
