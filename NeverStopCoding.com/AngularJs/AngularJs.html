Introduction to Angular Js Framework

Welcome to my Angular Javascript section. In this section we are going to talk about how to incorporate Angular Js into your HTML pages. In order to use Angular Js, you should be familar with HTML and Javascript.

The Basics

There are several things you should add before working with Angular Js in any type of html file. For one, you should add 'ng-app' to your html opening tag like so: '<html ng-app>' and you should also add the script in your head tag: '<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.1/angular.min.js"></script>'. You can also download the file from 'https://angularjs.org/' and then load the file on to your html similar to the way we added our jQuery file in the head tags. <script src="../angular.min.js"></script> Let's look at an example of storing data using angular JS.

Code: 
<!doctype html>
 
<html ng-app>
    <head>
        <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.1/angular.min.js">
        </script>
    </head>
        <body>  
            <div>  
                <label>Name:</label>
                <input type="text" ng-model="yourName" placeholder="Enter a name here">
                <br />
                <label>Hello {{yourName}}</label>
            </div>
        </body>
</html>

Applying Angular Js to elements

Let's say for example that we don't want to add Angular JS to our entire html file, we could add our 'ng-app' to a div element and then only items inside that element would be affected by it. Remeber, we still need to add our angular js script to the head tag reguardless of where we place our 'ng-app'.

Code: 
<div ng-app="">
    <input type="text" ng-model="yourName" placeholder="Enter a name here">
    <label>Your name: {{yourName}}<label>
</div>

Data Binding

We can also use data binding to bind our data to html elements, let's try an example using the 'p' element.

Code: 
    <div ng-app="">
        <p>Name: <input type="text" ng-model="name"></p>
    <p ng-bind="name"></p>
</div>
