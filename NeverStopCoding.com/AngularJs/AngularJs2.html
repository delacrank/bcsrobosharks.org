Controllers - Angular Js Development

Controllers

Similar to the Controller in an MVC pattern, a controller in Angular Js will manage the application layer of the program. Let's look at a sample controller to understand how it works.

Code: 
<div ng-app="myApp" ng-controller="myCtrl">

First Name: <input type="text" ng-model="firstName">
Last Name: <input type="text" ng-model="lastName">

Full Name: {{firstName + " " + lastName}}

</div>

<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
    $scope.firstName= "";
    $scope.lastName= "";
});
</script>

Let's understand what is going on here. The first thing you should notice is that we added a controller attribute to our div block element. Following this we gave our user to models in which to submit data. The first model is called 'firstName' and the second model is 'lastName'. Following this we used an expression to fill in a full name, similar to other languages we were able to concatect these two strings using the plus sign. Now let's look see how we were able to define our application by intializing a variable named app equal to angular.module('myApp', []); Let's move to our controller which we use to define the scope of our application. Controller Methods

Code: 
 
<div ng-app="myApp" ng-controller="myCtrl">

First Name: <input type="text" ng-model="firstName">
Last Name: <input type="text" ng-model="lastName">

Full Name: {{fullName}}

</div>

<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
    $scope.firstName= "";
    $scope.lastName= ""; 
    $scope.fullName = function() {
        return $scope.firstName + " " + $scope.lastName;
    }
});
</script>
